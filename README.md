Molecule_Game
=============

Html game created on MoleculeJS

```bash
git clone https://github.com/avil13/Molecule_Game.git
```

```bash
cd Molecule_Game
```

```bash
npm install
```

```bash
npm i -g gulp
```

```bash
gulp watch
```

Now You can change file  in coffee folder.

Or in ```js``` folder, but stop gulp. 
Because it is running only for ```coffee```  files
